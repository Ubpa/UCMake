# infra/.more-hooks/

git hook dispatcher 框架目录。解决多工具共用 git hook 时的注册冲突问题。

## 索引

| 名称 | 类型 | 职责 |
|---|---|---|
| `more-hooks.py` | 文件 | dispatcher 安装、hook 注册/注销、列表管理 |

## 架构作用

UCMake configure 时调用 `more-hooks.py install-to-project` 将本目录复制到下游项目，再调用 `register` 分别注册 codocs 和 UCMake 自己的 hook 脚本。

最终 `.git/hooks/pre-commit` 是 dispatcher，实际脚本在 `.git/hooks.d/pre-commit/` 下按优先级排列，互不覆盖。
