# infra/.ucmake/

UCMake 自身工具目录。cmake configure 时（`Ubpa_InitProject()`）将 hooks 注册到下游项目。

## 索引

| 名称 | 类型 | 职责 |
|---|---|---|
| `hooks/` | 目录 | UCMake git hook 静态脚本 |

## 设计说明

`.ucmake/` 与 `.codocs/` 平行，但职责不同：`.codocs/` 关注文档质量，`.ucmake/` 关注构建/测试质量。两者的 hook 脚本均通过 more-hooks dispatcher 共存于同一个 `pre-commit` 链中。

变量（build dir、config、project name）在 cmake configure 时写入 `.ucmake/project.env`，hook 静态脚本在运行时加载，实现脚本与项目配置的解耦。
